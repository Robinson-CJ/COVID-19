# covid_19_demo

## 目录结构
```
src
│
├── assets: 存放静态资源
│ 
│── component: 存放一般组件
│
│── plugins: 插件模块如 echarts
│
│── router: 路由模块
│
│── utils: 工具模块如 axios
│
│── views: 存放路由组件
│
│── App.vue
│── main.js
```
## Project setup
```
npm install
```

##  为 "src/ "目录路径起别名为 "@/"，方便后续导入模块

```
 "paths": {
      "@/*": ["src/*"]
    },
```

## Vue2 和 vue-router 的问题
因为 vue-router 已更新，最新版本只能在 Vue3 中使用，在 Vue2 中需要指明 vue-router@3 版本
npm install vue-router@3
